# 🚀 Ashish Suryavanshi — Portfolio

A modern, dark-themed personal portfolio website with smooth animations, floating navigation, and interactive UI elements.

![HTML](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)

## ✨ Features

- **Hero Section** — Eye-catching intro with animated SVG scribbles and social links
- **Featured Works** — Project showcase with desktop mockup-style cards
- **About Me** — Rotating SVG text ring and personal bio
- **Process Section** — Step-by-step workflow (Discovery → Design → Development)
- **Contact Section** — Quick access to email, GitHub, LinkedIn, and LeetCode
- **Floating Navigation** — Sticky nav bar that highlights the active section on scroll
- **Smooth Animations** — Scroll-triggered reveal animations throughout

## 🚀 Getting Started

### Prerequisites

No build tools or dependencies required — just a browser!

### Run Locally

1. **Clone the repo**

   ```bash
   git clone https://github.com/Exploitboy/portfolio.git
   ```

2. **Open in browser**

   ```bash
   cd portfolio
   open index.html
   ```

   Or simply double-click `index.html`.
